# Universal Church Network (UCN) 2026

Глобальная Web3-экосистема для объединения официальных церквей мира.

## 🏗 Архитектура проекта
- **/crawler**: Система сбора данных (OSM + Google Maps API + AI Email Extractor).
- **/contracts**: Смарт-контракты управления участниками (Solidity, Foundry).
- **/tma-app**: Telegram Mini App интерфейс для регистрации и взаимодействия.

## 🚀 Цели
1. Собрать базу данных >1 млн религиозных организаций.
2. Реализовать Onboarding через Account Abstraction (ERC-4337).
3. Создать систему верифицированных рассылок.

## 🛠 Стек технологий
- **Backend/Parser:** Python (Playwright, Beautiful Soup).
- **Blockchain:** Arbitrum / Base (L2 для низкого газа).
- **Frontend:** Next.js 15, Tailwind CSS.
